## 記号凡例

| 記号 | 意味 |
|------|------|
| `【】` | CADから取得する数量コード |
| `《》` | 部屋タイプ別集計 |
| `IF/THEN/ELSE` | 条件分岐 |
| `OR()` | いずれかがあれば1を返す |
| `ROUNDUP()` | 切り上げ |
| `ROUNDDOWN()` | 切り捨て |
