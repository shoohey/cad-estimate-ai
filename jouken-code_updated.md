## ⚠️ 2. 重要な注意事項（2026年2月追加）

### 2.1 CAD条件と実仕様の乖離について

**発見された問題:**
古居様邸の実績分析において、CADの特長条件設定と実際の見積仕様が一致しないケースが確認されました。

**具体例:**
- CADの条件01（R000051）= 1（鋼板屋根）
- 実際の見積 = 瓦屋根で計上

**原因として考えられること:**
1. CAD入力時と打合せ後の仕様変更
2. CAD設定の更新漏れ
3. 見積担当者による手動調整

**対応方針:**
- **CAD条件を鵜呑みにせず、必ず図面・打合せ記録を確認する**
- 条件設定と実仕様が異なる場合は、実仕様を優先
- 乖離が発見された場合は、CAD担当者にフィードバック

### 2.2 条件01の特殊な挙動

**鋼板屋根遮音下地について:**
通常、鋼板屋根遮音下地（ビルボード9mm）は条件01=1の場合のみ計上されるべきですが、実際の運用では**瓦屋根でも計上されるケースがある**ことが判明しました。

**理由:**
- 遮音性能を重視する施主の要望
- 標準仕様としての採用
- 見積テンプレートの設定

**AI見積システムでの対応:**
- 基本は条件01に従って自動判定
- 図面に遮音下地の記載がある場合は、条件01に関係なく計上
- 確認メッセージを出力して担当者に判断を促す

### 2.3 条件04と太陽光発電システム工事の関係

**従来の理解:**
条件04=1 → 電気工事内のHEMS・分電盤変更を計上

**実際の運用（更新後）:**
条件04=1 → 以下の両方を計上
1. 電気工事内のHEMS・分電盤変更
2. **独立した「太陽光発電システム工事」（19番工事）**

**注意:**
太陽光発電システム工事の金額は物件別に設定が必要です。パネル枚数、kW数、メーカーにより大きく変動します。

## 3. 条件別の影響範囲

### 条件01: 鋼板屋根

**影響する工事項目:**

| 工事種別 | 項目 | 条件01=1（鋼板） | 条件01=0（瓦） |
|----------|------|-----------------|---------------|
| 屋根工事 | 小屋裏換気（鋼板用） | 計上（1,200円/㎡） | 0 |
| 屋根工事 | 小屋裏換気（瓦用） | 0 | 計上（500円/㎡）★単価変更 |
| 屋根工事 | 鋼板屋根荷揚げ費 | 計上 | 0 |
| 屋根工事 | SGL鋼板たてひら葺 | 計上（7,100円/㎡） | 0 |
| 屋根工事 | 平板瓦葺 | 0 | 計上（7,200円/㎡）★単価変更 |
| 木工事 | 鋼板屋根遮音下地 | 計上 | 0（※注意事項参照） |

**計算式での使用例:**
```
IF 特長-条件01 = 1 THEN
  【屋根 面積】（鋼板用小屋裏換気）
ELSE
  0
```

**★2026年2月追記:**
鋼板屋根遮音下地は条件01=0でも計上される場合があります。図面の遮音仕様を確認してください。

---

### 条件02: 準防火地域

**影響する工事項目:**

| 工事種別 | 項目 | 条件02=1（準防火） | 条件02=0（一般） |
|----------|------|-------------------|-----------------|
| 換気システム | 屋外フードφ100（防火用） | 計上（2,900円/個） | 0 |
| 換気システム | 屋外フードφ100（一般用） | 0 | 計上（1,300円/個） |
| 換気システム | 屋外フードφ150（防火用） | 計上（8,600円/個） | 0 |
| 換気システム | 屋外フードφ150（一般用） | 0 | 計上（4,000円/個） |

**計算式での使用例:**
```
IF 特長-条件02 = 1 THEN
  便所数 + 浴室数 + 4（防火用フード）
ELSE
  0
```

**単価の違い:**
- 一般用フード: FP-100（金網無し）
- 準防火用フード: FPD-100（防火ダンパー付き）

---

### 条件03: IH採用

**影響する工事項目:**

| 工事種別 | 項目 | 条件03=1（IH） | 条件03=0（ガス） |
|----------|------|---------------|-----------------|
| 電気工事 | E付専用コンセント（IH用） | 計上 | 0 |
| 屋外付帯工事 | ガス工事 | 0 | 計上（物件別）★追加 |

**計算式での使用例:**
```
IH用コンセント数 = 特長-条件03 × OR(居間数, 台所数)
```

**備考:**
- IH用は200V専用回路が必要
- ガスの場合はガス工事で別途計上（物件別金額）

**★2026年2月追記:**
条件03=0（ガス採用）の場合、屋外付帯工事にガス工事を計上する必要があります。金額は敷地条件により変動するため、物件別に設定してください。

---

### 条件04: 太陽光発電

**影響する工事項目:**

| 工事種別 | 項目 | 条件04=1（太陽光あり） | 条件04=0（なし） |
|----------|------|----------------------|-----------------|
| 電気工事 | 太陽光対応分電盤変更 | 計上 | 0 |
| 電気工事 | スマートコスモ分電盤変更 | 計上 | 0 |
| 電気工事 | HEMS本体 | 計上 | 0 |
| 電気工事 | HEMS設定費 | 計上 | 0 |
| **太陽光発電システム工事** | **太陽光発電システム** | **計上（物件別）★追加** | 0 |

**計算式での使用例:**
```
HEMS本体 = 特長-条件04 × 1（台）
HEMS設定費 = 特長-条件04 × 1（式）
太陽光発電システム工事 = 特長-条件04 × 物件別金額
```

**関連設備:**
- AiSEG3（ゲートウェイ）
- スマートコスモ分電盤
- 太陽光パワーコンディショナー（別途）
- 太陽光パネル（19番工事で計上）

**★2026年2月追記:**
太陽光発電システム工事は独立した工事種別（19番）として計上します。電気工事内のHEMS・分電盤とは別計上のため、二重計上に注意してください。

## 6. CADでの設定方法

1. 物件マスタを開く
2. 「物件情報」タブを選択
3. 「特長」セクションで各条件にチェック
4. 保存して数量データを出力

**注意点（2026年2月更新）:**
- 条件の設定漏れに注意
- 見積作成前に必ず確認
- **打合せ後の仕様変更がある場合は、CADデータも更新する**
- **変更があった場合は再出力**
- **AI見積システムに投入する前に、条件設定と実仕様が一致しているか確認**

## 7. AI見積システムでの運用ルール（2026年2月追加）

### 7.1 条件判定のフロー

```
1. CADデータから条件コード（R000051〜R000054）を読み取る
2. 各条件に基づいて工事項目を自動選定
3. 以下の項目について確認メッセージを出力:
   - 条件01=1の場合: 「鋼板屋根で計算しています。瓦屋根の場合はご指摘ください。」
   - 条件04=1の場合: 「太陽光発電システム工事の金額を入力してください。」
   - 条件03=0の場合: 「ガス工事の金額を入力してください。」
```

### 7.2 乖離が発見された場合の対応

**AI見積システムが出力する警告:**
```
⚠️ 条件設定の確認が必要です:
- 条件01（鋼板屋根）= 1 ですが、建具データに瓦屋根用の部材が含まれています。
- 実際の仕様をご確認の上、手動で調整してください。
```

**担当者の対応:**
1. 図面・打合せ記録を確認
2. 実仕様に合わせて見積を手動調整
3. CAD担当者に条件設定の修正を依頼

### 7.3 物件別金額の入力が必要な項目

| 工事種別 | 項目 | 入力が必要な条件 |
|----------|------|-----------------|
| 19 太陽光発電システム工事 | 太陽光発電システム | 条件04=1 |
| 23 屋外付帯工事 | ガス工事 | 条件03=0 |
| 23 屋外付帯工事 | 屋外給排水工事 | 常時 |
| 23 屋外付帯工事 | 共通仮設工事 | 必要な場合 |
